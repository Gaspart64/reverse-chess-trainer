## Troubleshooting File Issues

**Missing manifest.json?**
- Check that it's in the same folder as the HTML
- Verify the file is valid JSON
- Check DevTools â†’ Application â†’ Manifest

**Service worker not registering?**
- Ensure sw.js is in the same folder as HTML
- Check for HTTPS (required for service workers)
- Look at DevTools â†’ Console for errors

**Paths not working?**
- Use relative paths (which this app does)
- Don't use absolute paths (won't work on GitHub Pages)
- All files must be in the root directory
